# BoviTrack - Plataforma de Seguimiento del Peso Bovino

## Descripción
Plataforma digital para el seguimiento del peso bovino y gestión de visitas veterinarias.

## Stack Tecnológico
- **Frontend**: React 18, TypeScript
- **Backend**: Node.js 20, TypeScript, AWS Lambda
- **Base de Datos**: Amazon RDS (PostgreSQL 15)
- **Infraestructura**: AWS CDK, API Gateway, S3, Docker
- **Autenticación**: Amazon Cognito

## Arquitectura
Arquitectura cloud-native en AWS con microservicios:
- Gestión de Ganado Service
- Seguimiento de Peso Service  
- Agenda Veterinaria Service
- Alertas y Notificaciones Service
